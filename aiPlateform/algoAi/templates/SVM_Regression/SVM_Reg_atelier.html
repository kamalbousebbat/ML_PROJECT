Atelier Pratique - Prédiction des calories brûlées

Dataset des gym membres

L'objectif de cet atelier est de montrer comment entraîner un modèle de régression supervisée avec Support Vector Regression (SVR) pour prédire le nombre de calories brûlées durant une séance de sport à partir des caractéristiques d'un utilisateur.

Le modèle entraîné et sauvegardé sera ensuite utilisé dans une application, où l'utilisateur pourra saisir les informations d'un membre via un formulaire et cliquer sur le bouton “Tester le modèle” pour obtenir une prédiction instantanée.
