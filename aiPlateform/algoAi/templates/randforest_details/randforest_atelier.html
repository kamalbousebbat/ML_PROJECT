Atelier Pratique – Prédiction du départ des employés

Dataset des employés

L’objectif de cet atelier est de montrer comment entraîner un modèle de classification superviséele Random Forest pour prédire si un employé quittera l’entreprise ou non à partir de ces caractéristiques.

Le modèle entraîné et sauvegardé sera ensuite utilisé dans une application, où l’utilisateur pourra saisir les informations d’un employé via un formulaire et cliquer sur le bouton “Tester le modèle” pour obtenir une prédiction instantanée.
